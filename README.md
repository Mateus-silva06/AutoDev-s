# AutoDev's

Bem-vindo ao repositório do **AutoDev's**! Este é um projeto de interface front-end para uma loja de carros moderna, criado para entregar uma experiência visual fluida e interativa para os usuários.

## 📖 Sobre o Projeto

O AutoDev's é um catálogo dinâmico de veículos. O grande destaque do site é o seu carrossel interativo: os usuários podem visualizar a foto do carro em destaque, junto com seu nome e informações detalhadas. Através de setas de navegação, é possível passar para o próximo veículo, o que dispara animações suaves e efeitos visuais imersivos, como uma sombra projetada logo abaixo do carro.

## ✨ Funcionalidades

* **Vitrine Interativa (Slider/Carrossel):** Navegação fluida entre os veículos disponíveis através de setas direcionais.
* **Exibição de Detalhes:** Apresentação limpa do nome e das especificações técnicas de cada carro.
* **Animações e Efeitos Visuais (UI/UX):** Transições animadas ao trocar de veículo e efeito de sombra dinâmica para dar profundidade e realismo às imagens.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando os pilares clássicos do desenvolvimento web:

* **[HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML):** Para a estruturação semântica do conteúdo e das imagens.
* **[CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS):** Responsável por todo o layout, efeitos de sombra (como `drop-shadow`) e as animações de transição entre os slides.
* **[JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) (Vanilla):** Utilizado para a lógica do carrossel, manipulação do DOM e controle dos eventos de clique nas setas.
